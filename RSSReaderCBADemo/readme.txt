Description


This application is to demo of using MVVM in Objective C. The purpose of this app is read data from rss source, including Daily Mail and Washington Post. It also uses KVO (Key Value Observing pattern) for data binding. Additionally, the app examinate the RSS2JSON API for parsing the RSS to JSON data. The view is created by Objective C programmytically by association of contrainst without storyboard of Xib file.

The application uses MagicalRecord library for store data and binding.

Some unit tests are also conducted in target bundle in order to evaluate the load data and save data  to local database.
