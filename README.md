## About The Project

This application is to demonstrate of using MVVM in Objective C. The purpose of this app is to read data from rss source, including Daily Mail and Washington Post. It also uses KVO (Key Value Observing pattern) for data binding. Additionally, the app examinates the RSS2JSON API for parsing the RSS to JSON data. The view is created by Objective C programmatically by an association of constraint without storyboard of Xib file.

The application uses MagicalRecord library for storing and binding data.

Some unit tests are also conducted in target bundle in order to evaluate the how data load and save into local database.

Use the `BLANK_README.md` to get started.

<!-- GETTING STARTED -->

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/HenryNguyenCDU/MVVMCBADemo.git
   ```
3. Install Cocoapods 
   ```sh
   npm install
   ```
4. Run ```pod install```
